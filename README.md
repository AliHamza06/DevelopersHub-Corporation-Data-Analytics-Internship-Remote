# DevelopersHub-Corporation-Data-Analytics-Internship-Remote


## Overview

This repository contains five Data Analytics and Machine Learning projects completed during my internship at DevelopersHub Corporation. The projects demonstrate practical applications of data preprocessing, exploratory data analysis, visualization, and predictive modeling using Python and Jupyter Notebook.

Each project includes:

* Dataset file
* Jupyter Notebook
* Data analysis and machine learning workflow

---

# Projects Included

## 1. Customer Churn Prediction

### Description

This project predicts customer churn using machine learning techniques. The workflow includes:

* Data cleaning
* Feature engineering
* Data visualization
* Model training
* Prediction analysis

### Files

* `Churn_FILE.ipynb`
* `Churn_csv.csv`

### How to Run

1. Download both files
2. Open `Churn_FILE.ipynb` in Jupyter Notebook
3. Install required libraries:

```bash
pip install pandas numpy matplotlib seaborn scikit-learn
```

4. Run all notebook cells
